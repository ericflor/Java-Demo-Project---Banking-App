Spark Bank Account User Manual

Opening An Account

  1. Click 'Add Accounts' Button
  2. Choose Account Type You Wish To Open From The Dropdown Menu
  3. Account Readout With New Account ID Will Printed Below

Note: Checking Accounts Start With Zero Balance & Have A $5 Overdraft Fee -- Savings Accounts Start With A $20.00 Balance & Have A Maximum of Three Monthly Withdrawals



Depositing Money

  1. Type In The Amount You Wish To Deposit
  2. On The Right-Hand Side Select 'deposit'
  3. Click the 'Execute' Button
  4. Return Value Will Be True If Executed Successfully

Note: Ensure The Account ID # You Wish To Deposit To Is Selected From The Left-Hand Dropdown Menu



Withdrawing Money

  1. Type In The Amount You Wish To Withdraw
  2. On The Right-Hand Side Select 'withdraw'
  3. Click the 'Execute' Button
  4. Return Value Will Be True If Executed Successfully

Note: Ensure The Account ID # You Wish To Withdraw From Is Selected From The Left-Hand Dropdown Menu



Overdraft Fees

    Your Checking Account will result in a $5 Overdraft Fee if the amount you are attempting to withdraw is greather than your current available balance.

    You will receive the following Error Notification and $5 will automatically be deducted from your account.



Maximum Monthly Withdrawals

    Your Savings Acccount is allowed a maximum of 3 withdrawals per month. If you attempt any more, you will be returned a false value and the deposit will not be executed.



Savings Balance Minimum

    Your Savings Account is required to have a minimum balance of $20.00. If you attempt to withdraw more you will receive the following notification and the withdrawal will not be executed.

Account Information

  1. Click 'View Accounts' Button
  2. Choose An Amount To Display All Accounts With The Specified Minimum Balance
  3. Choose Account Type From The Dropdown Menu To Display All Your Accounts With The Specified Type
  4. You Will Then Be Able To View A Printout Of All Accounts, The Number of Accounts, And The Accounts With the Parameters You Previously Selected


Note: You can see on Savings Account #2 that we have reached the maximum amount of monthly withdrawls.
